# Jobify

A modern full-stack Job Portal built using the MERN stack.

## 🚀 Overview

Jobify helps candidates discover job opportunities and allows companies to post and manage jobs through a clean and responsive interface.

This repository currently contains the frontend UI with job listing, search, sorting, filtering, and responsive layouts. Backend APIs and additional features are being integrated progressively.

---

## ✨ Features

- Responsive UI built with Tailwind CSS
- Job Listing Page
- Search Jobs
- Sort Jobs
- Filter by:
  - Work Mode
  - Employment Type
  - Salary
- Featured Jobs Section
- Company Listings
- Mobile Filter Drawer
- Responsive Navigation

---
